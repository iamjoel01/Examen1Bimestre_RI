## Instrucciones de ejecución (Google Colab)

1. Clonar este repositorio o descargar el archivo `.ipynb`.
2. Abrir el notebook en Google Colab.
3. Subir los archivos del corpus (`rotten_tomatoes_movies.csv` y `rotten_tomatoes_critic_reviews.csv`) a la carpeta `/content/sample_data/`.
4. Ir a `Entorno de ejecución` > `Cambiar tipo de entorno de ejecución` y seleccionar el acelerador **T4 GPU**.
5. Ejecutar la primera celda para instalar las dependencias o ejecutar `pip install -r requirements.txt`.
6. Ejecutar el notebook celda por celda secuencialmente.
